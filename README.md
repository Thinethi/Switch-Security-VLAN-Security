🔐 Cisco Switch Security Lab | VLAN Security & Layer 2 Attack Mitigation

As part of my hands-on networking practice, I completed a Cisco Packet Tracer lab focused on implementing Layer 2 switch security to protect enterprise networks against common attacks.

🛡️ Security Configurations Implemented

✅ Port Security

Configured port security on access ports.
Bound specific MAC addresses to each switch port.
Tested violation modes (Restrict) to prevent unauthorized devices from accessing the network.
Disabled all unused FastEthernet and GigabitEthernet ports to reduce potential attack surfaces.

✅ ARP Attack Mitigation

Configured a secure router-to-switch connection.
Prepared the network for ARP protection by implementing trusted Layer 2 connectivity, helping reduce the risk of ARP spoofing attacks.

✅ DHCP Snooping

Enabled DHCP Snooping on the switch.
Configured trusted and untrusted interfaces.
Applied DHCP rate limiting on access ports to defend against rogue DHCP servers and DHCP starvation attacks.

✅ VLAN Security & VLAN Attack Mitigation

Created separate VLANs for different departments.
Assigned access ports to the appropriate VLANs.
Configured secure trunk links using:
Manual trunking
Disabled DTP (switchport nonegotiate)
Restricted allowed VLANs
Configured the native VLAN
Implemented Router-on-a-Stick (Inter-VLAN Routing) using IEEE 802.1Q subinterfaces for secure communication between VLANs.
